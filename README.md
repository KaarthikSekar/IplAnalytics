## 📌 Overview

IPL Analytics is a data-driven web application that transforms raw IPL match data into meaningful insights through the power of **SQL queries** and **interactive visualizations**. Explore team performances, player statistics, season trends, and match outcomes — all in one place.

---

## 🚀 Features

- 🔍 **SQL-Driven Insights** — Every metric and visualization is powered by structured SQL queries
- 📊 **Interactive Dashboard** — Filter by season, team, player, and match type
- 🏆 **Team Performance** — Win/loss records, run rates, and head-to-head comparisons
- 🧑‍🤝‍🧑 **Player Statistics** — Batting averages, strike rates, wickets, and economy rates
- 📅 **Season Trends** — Year-by-year performance breakdowns
- ⚡ **Match Outcomes** — Toss decisions, venue impact, and result patterns

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend / UI | Streamlit |
| Backend Logic | Python |
| Data Layer | SQL (SQLite / Pandas SQL) |
| Visualization | Plotly / Matplotlib |
| Data Source | IPL Dataset (Kaggle) |

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ipl-analytics.git
cd ipl-analytics
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
